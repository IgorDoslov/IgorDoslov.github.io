Wasteland Wanderer

About:

Wasteland Wanderer is a 3rd person puzzle game set in an abandoned factory in a post-apocalyptic world. Make your way through a sprawling factory to discover the secrets hidden at the end.

Game:

  • 3rd Person Adventure/Puzzle game
  • Built in Unity
  • C# Programming

Role:

  • Building the Character Controller
  • Building the Event System
  • Building the Audio Manager
  • Collaborating with the team to create new mechanics
